# Cognix

Cognix is a premium AI prompt sharing and marketplace platform for discovering, publishing, bookmarking, reviewing, and monetizing high-quality AI prompts.

## Section 1 Status

This section establishes the monorepo foundation and Express API server. Later sections will add MongoDB models, authentication, marketplace APIs, dashboards, payments, and deployment configuration.

## Scripts

```bash
npm run dev:server
npm run dev:client
npm run build:client
```

## Stack

- Next.js App Router
- Tailwind CSS
- Node.js and Express.js
- MongoDB with Mongoose
- JWT authentication
- Stripe payments
